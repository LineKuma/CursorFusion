# NSIS Installer Assets

此目录存放 NSIS 安装包所需的资源文件。

## 所需文件

| 文件 | 说明 | 尺寸要求 |
|------|------|----------|
| `icon.ico` | 应用程序图标（安装包图标、桌面快捷方式） | 256x256 或多尺寸 ICO |
| `header.bmp` | 安装向导头部图片 | 150x57 像素，24位色 |
| `wizard.bmp` | 欢迎页/完成页侧边图片 | 164x314 像素，24位色 |

## 获取方式

1. **从项目 logo 生成**：使用 [icoconvert](https://convertico.com) 等工具将 PNG 转 ICO
2. **设计工具**：使用 Figma、Photoshop 等创建 BMP 图片
3. **占位图片**：可暂时使用纯色或渐变背景的占位图

## 注意

- 请确保替换为实际的资源文件后再触发构建
- 如果缺少文件，NSIS 编译会报错
